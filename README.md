# 📱 爸妈助手 - Parent Assistant

**爸爸妈妈的智能手机助手**，帮助他们更轻松地使用智能手机，解决日常生活中的小麻烦。  
通过简化操作和自动化任务，让父母在使用手机时更加便捷、舒适、安全。

开发初衷是父母不怎么会用手机和认字，所以我想完成一些功能，比如地图导航，打开收款码，快速打电话，等。
